## 🦊 Authority Delegation Rules

1. Each agent commands **exactly 3 subordinates** (except leaf nodes)
2. Commands flow down, reports flow up
3. Luna Bytefox has final authority on all decisions
4. Cross-branch coordination requires approval from common superior
5. Emergency protocols: skip one level up if immediate superior unavailable
