# Turbo Project Resource & Asset Hub

## Project Overview
This directory serves as a centralized **Knowledge Base and Resource Hub** for experimentation with automation, AI research, and technical documentation. It is not a single software project but rather a curated collection of data, tools, and research materials.

The workspace is heavily focused on:
- **Brainrot & Neural Research:** AI-driven philosophical exploration (e.g., "Enlightened Neural Networks").
- **Locksmithing & Safe Automation:** Documentation and code for automated safe dialers (Arduino-based).
- **API Research:** Comprehensive API specifications (e.g., Canva Connect API).
- **AI Exploration:** Reports, prompts, and history related to Google AI Studio and general AI trends.
- **Electrical Theory:** NEC (National Electrical Code) resources.
- **Personal Data Archiving:** Reddit Vault backups, spreadsheet-based tracking, and document templates.

## Directory Overview

### 1. **Brainrot & Neural Research (`brainrot.*`, `manifesto.md`)**
- Core logic and documentation for "The Infinite Digital Dopamine Loop" and AI enlightenment experiments.
- **`brainrot.ts/tsx`**: Interactive components for neural simulations.
- **`brainrot.md`**: Theoretical framework and "Enlightened Neural Network" tracking.

### 2. **Locksmithing & Security (`Files/Pdf/`, `pdf/`, `txt/`)**
- Extensive collection of manuals for LaGard, S&G, and other high-security lock systems.
- **`txt/main.cpp.txt`**: Arduino source code for a stepper-motor-driven automated safe dialer.
- **`Files/Pdf/Safe-Autodialer · GitHub.pdf`**: Reference for the autodialer project.

### 2. **AI & Technology Research (`pdf/`, `Files/Google AI Studio/`)**
- **AI Policy & Reports:** Documents like `Student AI Policy 2025.approved.pdf` and `hai_ai_index_report_2025.pdf`.
- **Prompts:** Collection of prompts and access history in `Files/Google AI Studio/`.
- **API Specs:** `yml/api.yml` contains the full OpenAPI spec for the Canva Connect API.

### 3. **Data & Documentation (`docs/`, `sheets/`, `csv/`)**
- **Strategy Documents:** Business and strategy documents like `Forbes List Generation Strategy.docx`.
- **Spreadsheets:** Tracking for various projects, including `TurboTheTurtle.xlsx` and `Cheat sheets .xlsx`.
- **Templates:** Word templates for character letters and indexing.

### 5. **Absolute Velocity & Automation Research (`absolute_velocity.json`, `master_surge.sh`)**
- **`absolute_velocity.json`**: A collection of 10 "Absolute Velocity" deployment scenarios for high-volume repository automation and stress-testing.
- **`master_surge.sh`**: Script for executing parallelized repository interactions.
- **`publish_issues.sh`**: Utility for bulk-publishing issues to GitHub.

## Usage
- **Research:** Use the `pdf/` and `Files/Pdf/` directories to find technical specifications for locks or electrical code.
- **Development:** Refer to `txt/main.cpp.txt` for automation logic or `yml/api.yml` for Canva integration.
- **Data Management:** Utilize `sheets/` for ongoing project tracking and `docs/` for strategy formulation.

**Note:** This directory is an archive. New resources should be placed in the appropriate subdirectory based on file type or category.
