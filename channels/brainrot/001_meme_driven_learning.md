## Why Brainrot Works (The Science Nobody Asked For)

### 1. Emotional Anchoring

Your brain remembers emotions stronger than facts.

**Traditional:**
```python
# Textbook: "Functions encapsulate reusable logic"
def add(a, b):
    return a + b
```
Boring. Forgot it already.

**Brainrot:**
```python
# "Functions are just spicy variables that do tricks"
def yeet_numbers(a, b):
    return a + b  # no cap fr fr
```

Memorable because:
- "Spicy variables" creates mental image
- "Yeet" triggers cultural context
- "No cap fr fr" adds humor
- You'll never forget what a function does

## The Ohio Principle (Why Absurdity Works)

**Ohio** in brainrot culture = Chaos. Unpredictability. The place where rules break.

**Applied to learning:**

Traditional: "Follow these 10 steps exactly."

Brainrot: "Here's the concept. Now break it. See what happens. That's Ohio."

**Example:**

Traditional regex tutorial:
```
Step 1: Learn metacharacters
Step 2: Learn quantifiers
Step 3: Learn groups
[Continue for 20 steps]
```

Brainrot regex tutorial:
```
1. Try to match emails with: .*@.*
2. Watch it match "a@b" (technically correct!)
3. Realize you need to be more specific
4. Try: \w+@\w+\.\w+
5. Watch it fail on "name+tag@site.co.uk"
6. Realize email regex is Ohio-tier chaos
7. Give up and use a library

Congrats, you learned regex by breaking it.
```
