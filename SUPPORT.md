# Support Guide

If you need help with anything in this repository, here are the best ways to get it:

## Documentation
- Check the `README.md` for general information.
- Refer to `GEMINI_BRAINROT.md` for the project hub overview.
- Look into the specific subdirectories for technical manuals (e.g., `pdf/` for lock manuals).

## Issues
- If you find a bug or have a feature request, please [open an issue](https://github.com/Turbo-the-tech-dev/brainrot/issues).
- For security vulnerabilities, please refer to the [SECURITY.md](SECURITY.md).

## Community
- Join our discussions on [GitHub Discussions](https://github.com/Turbo-the-tech-dev/brainrot/discussions) if available.
- For more direct inquiries, reach out to the project maintainer.

Thank you for being part of the Turbo Project!
